# Jonathan Curtis Personal Portfolio

A responsive one-page personal portfolio for Jonathan Curtis, built as a static site suitable for GitHub Pages.

## Pages

- `index.html` — portfolio homepage with about, experience, education, leadership, skills, and contact sections.
- `resume.html` — web-friendly resume page based on the supplied resume.

## Publishing on GitHub Pages

1. Push this repository to GitHub.
2. Open repository **Settings → Pages**.
3. Select **Deploy from a branch**, choose the current branch, and use the repository root (`/`) as the source.
4. Visit the generated GitHub Pages URL and verify that `index.html`, `resume.html`, and `Jonathan Senior Photo.jpeg` load correctly on desktop and mobile.
